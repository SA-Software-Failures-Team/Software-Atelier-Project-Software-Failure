USB and LNK Exploit

Stuxnet spread initially via infected USB drives. It used a malicious .LNK shortcut exploit that executed automatically when viewed in Windows Explorer, enabling the worm to cross air-gapped networks.


Spooler Vulnerability (MS10-061)
Another vector exploited the Windows Print Spooler service to allow remote or local file writes with system privileges, helping Stuxnet propagate through corporate or plant networks.


Pilegeriv Escalation and Persistence Used multiple kernel-level privilege escalation vulnerabilities. Employed stolen digital certificates from Realtek and JMicron for driver signing to appear legitimate.


Command & Control (C2) Servers Stuxnet connected via HTTP to specific hardcoded domains, uploading telemetry and receiving updates. Analysis indicated most infections occurred in Iran, but spread was observed in over 150 countries.
