Root Cause and Technical Analysis


WannaCry exploited a critical Windows vulnerability known as EternalBlue, which affected the SMBv1 protocol. Originally developed by the NSA and leaked online, this exploit allowed remote code execution on unpatched systems. Computers exposed to the internet or connected to the same network were automatically targeted.

The malware deployed a backdoor tool called DoublePulsar, enabling further network propagation and remote control of infected machines. Once installed, WannaCry encrypted users’ files using AES-128 and RSA encryption, appending a .WNCRY extension and displaying a ransom note demanding Bitcoin payment.

A hidden “kill switch” domain within the malware was discovered by a security researcher, temporarily halting its spread. Despite this, the attack rapidly infected thousands of organizations worldwide due to unpatched systems and flat network architectures.
